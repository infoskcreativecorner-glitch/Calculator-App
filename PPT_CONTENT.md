## Slide 2: Introduction
- A calculator is an essential digital tool for quick arithmetic operations.
- Web-based applications are lightweight, accessible, and platform-independent.
- This project develops a browser-based calculator using core web technologies.
- Focus areas: usability, clean UI, and reliable error handling.

## Slide 3: Problem Statement
- Users need quick calculations without installing software.
- Many basic calculator projects fail on invalid input handling.
- Goal: build a simple but robust calculator with a modern interface.
- Required support: +, -, *, /, clear function, and safe evaluation.

## Slide 4: Objectives
- Develop calculator using HTML, CSS, and JavaScript.
- Implement basic arithmetic operations.
- Add clear, backspace, and keyboard input support.
- Handle invalid expressions and division-by-zero errors.
- Provide responsive design for desktop and mobile.

## Slide 5: Technology Used
- **HTML5**: Structure of calculator and buttons.
- **CSS3**: Styling, grid layout, responsive UI.
- **JavaScript (ES6)**: Event handling and expression evaluation.
- Development tools: VS Code, Chrome/Edge browser.

## Slide 6: System Design
- Three-layer architecture:
  - Presentation Layer (HTML)
  - Style Layer (CSS)
  - Logic Layer (JavaScript)
- Modules:
  - Input Handling
  - Expression Evaluation
  - Error Handling
  - Display Update
- Theme based on dark background with cyan accents.

## Slide 7: Features
- Basic operations: Addition, Subtraction, Multiplication, Division.
- Parentheses and decimal input support.
- Clear (C) and backspace (⌫) functions.
- Real-time display updates.
- Keyboard and mouse interaction.
- Error messages for invalid calculations.

## Slide 8: Demo Explanation
- Open `index.html` in browser.
- Enter expression using buttons or keyboard.
- Press `=` to view result.
- Try invalid input to observe error handling.
- Use `C` to reset and `⌫` to remove last character.
- Demonstrate mobile responsiveness by resizing browser.
