# Submission Guide - Calculator Application Package

## Package Contents

- `index.html` - calculator structure
- `style.css` - themed UI styling
- `script.js` - calculator logic and error handling
- `PROJECT_REPORT.md` - full project report draft
- `PPT_CONTENT.md` - 10-slide presentation content
- `DEMO_VIDEO_SCRIPT.md` - voice-over friendly demo script
- `VIVA_QA.md` - 20 viva questions with answers
- `BONUS_VARIATIONS.md` - alternate titles and UI options

## Quick Usage

1. Run app: open `index.html` in browser.
2. Convert report to PDF:
   - Open `PROJECT_REPORT.md` in Word/Docs or Markdown editor.
   - Add your college details and export as PDF.
3. Prepare PPT:
   - Copy `PPT_CONTENT.md` slide-wise into PowerPoint/Canva.
4. Demo recording:
   - Read from `DEMO_VIDEO_SCRIPT.md` while showing live app.
5. Viva prep:
   - Practice from `VIVA_QA.md`.
