# Demo Video Script - Calculator Application

## 1. Opening (20-30 seconds)

This project is a browser-based calculator designed to perform basic arithmetic operations with a clean user interface and proper error handling.

## 2. Project Overview (30-40 seconds)
The main objective of this project is to create a simple, responsive, and user-friendly calculator.  
It supports addition, subtraction, multiplication, and division.  
Along with that, I have included clear and backspace buttons, keyboard support, and handling for invalid expressions.

## 3. Interface Walkthrough (40-50 seconds)
Now I will show the interface.  
At the top, we have a display area where the expression and result are shown.  
Below that, there are numeric buttons from 0 to 9, decimal point, operators, parentheses, clear button, backspace button, and equals button.  
The color theme is dark with cyan highlights for better visibility and modern appearance.

## 4. Feature Demonstration (2-3 minutes)
Now I will demonstrate the features:

1. **Basic Addition**  
I enter 25 plus 17 and press equals.  
The result displayed is 42.

2. **Subtraction**  
I enter 50 minus 18 and press equals.  
The result is 32.

3. **Multiplication**  
I enter 9 multiplied by 7 and press equals.  
The output is 63.

4. **Division**  
I enter 100 divided by 5 and press equals.  
The result is 20.

5. **Brackets and Decimal**  
Now I enter open bracket 3 plus 2 close bracket multiplied by 4.  
The output is 20.  
I can also enter decimal numbers like 7.5 plus 2.5, and it returns 10.

6. **Clear and Backspace**  
If I press C, the display resets to zero.  
If I type 123 and press backspace, it removes the last digit and becomes 12.

7. **Error Handling**  
Now I test an invalid input, for example 8 divided by 0.  
The application shows an error message instead of crashing.  
If I type an incomplete expression like 5 plus and press equals, it again shows an invalid expression message.

8. **Keyboard Support**  
I can also type numbers and operators directly from the keyboard and press Enter for result.

## 5. How It Works (45-60 seconds)
This application is built using three files:

- `index.html` for the layout and calculator buttons.
- `style.css` for colors, spacing, and responsive design.
- `script.js` for calculator logic.

JavaScript captures button clicks and keyboard inputs, builds the expression, validates it, evaluates the result, and then updates the display.  
If the expression is invalid, it catches the error and shows a message.
