# University-Style DOCX Template Blueprint

This file gives you a strict formatting structure you can apply in MS Word to produce a professional final-year mini project report.

## 1. Document Setup

- Paper size: A4
- Margins: 1 inch on all sides
- Font (body): Times New Roman, 12 pt
- Line spacing: 1.5
- Paragraph alignment: Justified
- Paragraph spacing: Before 0 pt, After 6 pt
- First line indent: 0.5 inch (for normal paragraphs)

## 2. Heading Styles

- Heading 1: 16 pt, Bold, UPPERCASE, center/left as required
- Heading 2: 14 pt, Bold
- Heading 3: 12 pt, Bold Italic
- Use automatic styles for TOC generation

## 3. Page Numbering Format (Important)

### Section A: Preliminaries
- Includes: Title, Certificate, Declaration, Acknowledgement, Abstract, Table of Contents
- Page numbering style: Roman (i, ii, iii, iv...)
- Position: Bottom center
- Title page number hidden (but counted)

### Section B: Main Report
- Starts from Introduction
- Page numbering style: Arabic (1, 2, 3...)
- Restart numbering at 1
- Position: Bottom center

In Word:
1. Insert Section Break (Next Page) before Introduction.
2. Unlink header/footer from previous section.
3. Apply Roman numbering in Section A.
4. Apply Arabic numbering in Section B and set start at 1.

## 7. Table and Figure Formatting

- Table title: Above table (e.g., Table 1.1: Test Cases)
- Figure title: Below figure (e.g., Figure 3.1: Calculator Interface)
- Center align all table/figure titles
- Keep consistent numbering per chapter/section

## 8. Reference Style (Simple IEEE-like)

Use numbered references:

[1] MDN Web Docs, "HTML: HyperText Markup Language," Available: https://developer.mozilla.org/docs/Web/HTML  
[2] MDN Web Docs, "CSS: Cascading Style Sheets," Available: https://developer.mozilla.org/docs/Web/CSS  
[3] MDN Web Docs, "JavaScript," Available: https://developer.mozilla.org/docs/Web/JavaScript

## 9. Final DOCX Checklist

- All placeholders replaced with your details
- Uniform fonts and spacing
- TOC updated (References -> Update Table)
- Roman + Arabic numbering correctly applied
- No spelling/grammar highlights
- Export as PDF for final submission
