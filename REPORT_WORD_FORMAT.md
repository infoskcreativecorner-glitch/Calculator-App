## ABSTRACT

This project presents the design and development of a web-based Calculator Application using HTML, CSS, and JavaScript. The application performs basic arithmetic operations including addition, subtraction, multiplication, and division. It includes a clear function, backspace feature, support for decimal values and parentheses, keyboard input handling, and error management for invalid expressions.

The project follows a modular architecture by separating structure, style, and logic into different files. The user interface is designed with a dark theme and cyan highlights inspired by the provided brand image, ensuring readability and modern aesthetics.

Testing confirms accurate output for valid expressions and graceful handling for errors such as division by zero and malformed input. The developed application is lightweight, responsive, and suitable for academic demonstration.

Keywords: Calculator, HTML, CSS, JavaScript, Web Application, Error Handling, UI Design

## 1. INTRODUCTION

Calculators are fundamental digital tools used in education and daily tasks. A web-based calculator is highly useful because it runs directly in a browser without installation.

This project develops a browser-based arithmetic calculator with a clean interface and robust input handling. The design focuses on usability, responsiveness, and reliability. The application demonstrates practical knowledge of front-end development and software engineering documentation.

## 2. PROBLEM STATEMENT

Users often require quick calculation tools while working online. Basic calculator implementations commonly fail in handling invalid expressions and edge cases.

The problem addressed in this project is to create a simple but dependable calculator that supports core arithmetic operations with proper validation and user-friendly feedback.

## 3. OBJECTIVES

- To build a calculator app using HTML, CSS, and JavaScript.
- To implement +, -, *, and / operations.
- To provide clear and backspace functionality.
- To handle invalid input and division-by-zero safely.
- To design a clean responsive interface.
- To prepare complete academic documentation for submission.

## 4. SCOPE OF PROJECT

Out of Scope:

## 5. LITERATURE REVIEW

Software calculators evolved from desktop tools to browser-based utilities. Existing web calculators show the importance of readability, clear controls, and reliable validation. Many beginner implementations overlook malformed input handling. This project addresses those gaps through sanitization and guarded evaluation.
